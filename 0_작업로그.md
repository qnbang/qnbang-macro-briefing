# 매크로 투자 브리핑 서비스 — 작업로그

> 거시 흐름을 먼저 읽고 "이 자산을 이런 관점으로 보라"를 기사 형식으로 자동 발행하는 서비스.
> 포지셔닝: 매수/매도 신호가 아닌 **거시 흐름 학습 + 시나리오 관찰 도구**.
> 차별화: 글로벌 거시 흐름을 **한국 투자자의 자산 언어(원화·코스피·수출산업)로 번역**.

## 2026-06-11 — 기획 검토·보강 (v0.1 → v0.2)
- 사용자가 v0.1 기획서 제출 → 비판적 검토·보강 진행.
- 핵심 보강 6가지:
  1. **숫자/서사 분리** — 거시 수치는 구조화 데이터 API에서만(할루시네이션 1순위 차단), 구조화 데이터를 MVP 핵심으로 승격.
  2. **3중 검증 게이트** — 수집 화이트리스트 → 제약 생성(숫자 추가 금지) → 검수 패스(팩트 대조).
  3. **시나리오 DB 스키마 확정** — 트리거/반증/파급체인/상태/이력로그. 적중 추적을 1일차부터 데이터로 축적.
  4. **모델·비용 배분** — 일간 경량(Haiku/Sonnet), 월간 이상 Opus. 비용 가드레일.
  5. **포지셔닝 명확화** — 학습·관찰 도구(유사투자자문 규제 방어선 겸 정체성).
  6. **MVP 순서 재조정** — 시나리오 DB → 구조화 데이터 → 이벤트 → 분석체인 → 기사+검수.
- 산출물: [2_기획/기획서-v0.2.md](2_기획/기획서-v0.2.md)
- **구동·비용 결정**: 토큰당 API 결제 대신 **이미 내는 클로드 구독 + 클로드 코드 자동 실행(headless)**으로 → 추가 비용 0원, Opus 품질 유지. 비용 상한 항목 폐기.
- 다음 결정 대기: 데이터 소스(FRED/ECOS 키), 시나리오 시드 주제, 코딩 착수 단계.
